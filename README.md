# Dotfiles

Dotfiles for my ubuntu machine

## Requirements
- **terminal** urxvt
- trash-cli (replaces rm)
- neovim:
  - plug
  - nvim (npm package)
- i3
  - i3-lock-fancy
  - the script suspend.sh to be located in ~/bin
- Bumblebee-status requires:
  - FontAwesome
  - python-tk
  - python-psutil
  - python-netifaces
