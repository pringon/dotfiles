# Dotfiles

Dotfiles for my personal arch machine

# Usage

Drop dotfiles using `stow .`

## Requirements

- **terminal** urxvt:
  - Inconsolata powerline font
- neovim:
  - plug
  - nvim (npm package)
- i3
  - the script suspend.sh to be located in ~/bin
  - `rofi` must be installed
- Bumblebee-status requires:
  - FontAwesome
  - python-tk
  - python-psutil
  - python-netifaces
